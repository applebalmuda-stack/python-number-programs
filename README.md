# Python Number Programs

This repository contains beginner-friendly Python number programs and mathematical exercises.

## Programs Included

- Factorial
- Prime Number Check
- Fibonacci Series
- Armstrong Number
- Palindrome Number
- Even or Odd
- Sum of Digits
- Multiplication Table
- Largest of Three Numbers
- Leap Year Check

## Purpose

These programs help beginners practice Python number concepts, loops, conditions, and problem-solving skills.
